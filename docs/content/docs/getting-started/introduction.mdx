Invect is a workflow orchestration library that mounts directly into your existing Node.js application. It provides a visual flow editor, AI agent nodes, 190+ built-in integrations, and batch processing вҖ” all as npm packages, not a separate platform.

## How it works

Install the packages. Mount a router. Open the visual editor. Build workflows that connect your AI models, APIs, and data вҖ” then trigger them from your code or on a schedule.

```ts title="index.ts"
import express from 'express';
import { createInvectRouter } from '@invect/express';

const app = express();
const router = await createInvectRouter({
  database: { type: 'sqlite', connectionString: 'file:./dev.db' },
  encryptionKey: process.env.INVECT_ENCRYPTION_KEY!,
});
app.use('/invect', router);
app.listen(3000);
```

That's it. You now have:

- A visual flow editor at `http://localhost:3000/invect`
- A full REST API for managing and executing workflows
- AI agent support with tool calling
- 190+ integration nodes ready to use

## Architecture
