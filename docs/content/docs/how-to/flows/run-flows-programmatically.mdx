## Using the Invect core instance

If you have access to the `Invect` core instance (via dependency injection or direct reference), you can run flows directly:

```ts
const result = await invect.startFlowRun(flowId, {
  name: 'Alice',
  topic: 'machine learning',
});

console.log(result.status);   // 'COMPLETED' or 'FAILED'
console.log(result.output);   // Flow output data
```

### In Express

```ts
import { createInvectRouter } from '@invect/express';
import { invectConfig } from './invect.config';

const router = await createInvectRouter(invectConfig);
// The router's core instance is not directly exposed.
// Use the REST API instead, or access Invect core directly:

const { Invect } = await import('@invect/core');
const invect = new Invect(invectConfig);
await invect.initialize();

const result = await invect.startFlowRun(flowId, inputs);
```

### In NestJS

Inject `InvectService` in your own services:

```ts
import { Injectable } from '@nestjs/common';
import { InvectService } from '@invect/nestjs';

@Injectable()
export class MyService {
  constructor(private readonly invectService: InvectService) {}

  async runWorkflow(flowId: string, inputs: Record<string, unknown>) {
    const core = this.invectService.getCore();
    return core.startFlowRun(flowId, inputs);
  }
}
```

## Using the REST API

From any language or service, trigger a flow via HTTP:

```bash
curl -X POST http://localhost:3000/invect/flows/{flowId}/run \
  -H "Content-Type: application/json" \
  -d '{ "inputs": { "name": "Alice", "topic": "machine learning" } }'
```

Response:

```json
{
  "id": "run_abc123",
  "flowId": "flow_xyz",
  "status": "COMPLETED",
  "output": { ... },
  "startedAt": "2024-01-01T00:00:00.000Z",
  "completedAt": "2024-01-01T00:00:05.000Z"
}
```

## Listing flow runs

```bash
# All runs for a flow
curl http://localhost:3000/invect/flows/{flowId}/runs

# A specific run
curl http://localhost:3000/invect/flow-runs/{runId}
```
