The Chat Assistant is a built-in AI interface that helps you build and modify flows using natural language.

## Accessing the assistant

Click the **Chat** icon in the sidebar to open the chat panel. The assistant connects to your configured LLM credentials.

## Setup

The chat assistant requires:

1. An LLM API key credential (OpenAI or Anthropic) stored in Invect.
2. The credential selected in the chat panel's settings.

Open the chat panel and use the **provider selector** and **model selector** at the top to choose your credential and model.

## What it can do

- **Describe a flow** — "Create a flow that fetches weather data and sends a Slack notification."
- **Modify existing flows** — "Add an if/else node to check if the temperature is above 30°C."
- **Explain nodes** — "What does this HTTP Request node do?"
- **Debug issues** — "Why is my flow failing at the Template String node?"
