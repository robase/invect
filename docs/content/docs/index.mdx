Invect is an open-source workflow orchestration library that you mount directly into your existing Node.js application. Visual flow editor, AI agents, 190+ integrations, and batch processing — all as npm packages, not a separate platform.

## Get started in one command

```bash
npx invect-cli init
```

The CLI detects your framework, installs the right packages, creates your config, and sets up the database. Or see the framework-specific code below.

## Quick look

<Tabs items={['Express', 'NestJS', 'Next.js']}>
  <Tab value="Express">
  ```ts title="index.ts"
  import express from 'express';
  import { createInvectRouter } from '@invect/express';

const app = express();
const router = await createInvectRouter({
database: { type: 'sqlite', connectionString: 'file:./dev.db' },
encryptionKey: process.env.INVECT_ENCRYPTION_KEY,
});
app.use('/invect', router);
app.listen(3000);

````
</Tab>
<Tab value="NestJS">
```ts title="app.module.ts"
import { Module } from '@nestjs/common';
import { InvectModule } from '@invect/nestjs';

@Module({
  imports: [
    InvectModule.forRoot({
      database: { type: 'sqlite', connectionString: 'file:./dev.db' },
      encryptionKey: process.env.INVECT_ENCRYPTION_KEY,
    }),
  ],
})
export class AppModule {}
````

  </Tab>
  <Tab value="Next.js">
  ```ts title="app/api/invect/[...invect]/route.ts"
  import { createInvectHandler } from '@invect/nextjs';

const handler = createInvectHandler({
database: { type: 'sqlite', connectionString: 'file:./dev.db' },
encryptionKey: process.env.INVECT_ENCRYPTION_KEY,
});

export const GET = handler.GET;
export const POST = handler.POST;
export const PATCH = handler.PATCH;
export const PUT = handler.PUT;
export const DELETE = handler.DELETE;

```
</Tab>
</Tabs>

That's it. You have a visual flow editor and a full REST API for managing and executing workflows.
