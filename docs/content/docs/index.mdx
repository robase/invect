Invect is an open-source workflow orchestration library that you mount directly into your existing Node.js application. It provides a visual flow editor, AI agent nodes, 180+ built-in integrations, and batch processing — all as npm packages, not a separate platform.

## How it works

Install the packages. Mount a router. Open the visual editor. Build workflows that connect your AI models, APIs, and data — then trigger them from your code or on a schedule.

```ts
import express from 'express';
import { createInvectRouter } from '@invect/express';

const app = express();
const router = await createInvectRouter({
  database: { type: 'sqlite', connectionString: 'file:./dev.db' },
});
app.use('/invect', router);
app.listen(3000);
```

That's it. You now have a visual flow editor at `http://localhost:3000/invect` and a full REST API for managing and executing workflows.
